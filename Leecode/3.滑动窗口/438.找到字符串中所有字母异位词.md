给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

**示例 1:**

**输入:** `s = "cbaebabacd", p = "abc"`
**输出:** `[0,6]`
**解释:**
起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。

 **示例 2:**

**输入:** `s = "abab", p = "ab"`
**输出:** `[0,1,2]`
**解释:**
起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。

```Python
class Solution:wd 
    def findAnagrams(self, s: str, p: str) -> List[int]:
        list_s=list(s)
        list_p=sorted(list(p))
        s_n=len(list_s)
        p_n=len(list_p)
        ans=[]
        
        for left in range(s_n-p_n+1):
            if sorted(list_s[left:(left+p_n)])==list_p:
                ans.append(left)
        return ans
```

```Python
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        if len(p) > len(s):
            return []

        from collections import Counter # 用来统计元素出现次数,键是元素，值是出现次数。

        p_count = Counter(p)
        window = Counter()
        res = []

        left = 0
        for right, ch in enumerate(s):
            window[ch] += 1

            # 如果窗口长度超过 p
            if right - left + 1 > len(p):
                window[s[left]] -= 1 # 把窗口左端的字符出现次数减1
                if window[s[left]] == 0: # 如果计数变成 0，则删除该键
                    del window[s[left]]
                left += 1

            # 判断是否匹配
            if window == p_count:
                res.append(left)

        return res
```
