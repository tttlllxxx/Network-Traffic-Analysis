给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**示例 1:**

**输入:** `strs = ["eat", "tea", "tan", "ate", "nat", "bat"]`

**输出:** `[["bat"],["nat","tan"],["ate","eat","tea"]]`

**解释：**

- 在 strs 中没有字符串可以通过重新排列来形成 `"bat"`。
- 字符串 `"nat"` 和 `"tan"` 是字母异位词，因为它们可以重新排列以形成彼此。
- 字符串 `"ate"` ，`"eat"` 和 `"tea"` 是字母异位词，因为它们可以重新排列以形成彼此。

**示例 2:**

**输入:** `strs = [""]`

**输出:**` [[""]]`

**示例 3:**

**输入:** `strs = ["a"]`

**输出:**` [["a"]]`


```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        # defaultdict(list)表示字典的默认值是一个空列表
        # 访问一个不存在的 key 时，自动创建一个空列表
        mp=defaultdict(list) 
        for str in strs:
            key="".join(sorted(str))
            mp[key].append(str)
        return list(mp.values())
```

```Python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        sortlist={}
        for str in strs:
            key="".join(sorted(str))
            if key not in sortlist:
                sortlist[key]=[]
            sortlist[key].append(str)
        return list(sortlist.values())
```
