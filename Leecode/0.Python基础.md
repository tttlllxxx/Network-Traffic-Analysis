## 1.容器

Python 有四个最常用的容器：

|   容器    | 是否可变  |   是否可重复    |   是否有顺序   |      常用场景      |
| :-----: | :---: | :--------: | :-------: | :------------: |
| list[]  | ✔ 可变  |   ✔ 可重复    |   ✔ 有顺序   |   最常用：数组、队列    |
| tuple() | ✘ 不可变 |   ✔ 可重复    |   ✔ 有顺序   | 函数返回多值、不可变 key |
| dict{}  | ✔ 可变  | ✔ key 不可重复 | ✔ key 无顺序 |      哈希表       |
|  set{}  | ✔ 可变  |   ✘ 不可重复   |   ✘ 无顺序   |    去重、快速查找     |

### List\[ ]（列表）

创建
```Python
arr = [1,2,3]
```
获取元素
```Python
arr[0] # 1
arr[1] # 2
```
**常用操作**
```Python
arr.append(x)      # 末尾添加
arr.extend([3,4])  # 末尾添加[3,4]
arr.pop()          # 删除最后一个
arr.pop(i)         # 删除第 i 个
arr.remove(x)      # 删除list中的第一个x
arr.index(x)       # x第一次出现的位置
arr.count(x)       # 统计x出现的次数
arr.sort()         # 原地排序
arr.reverse()      # 原地反转
arr.insert(2,3)    # 在索引2的位置插入元素3
```
切片 slice
```Python
arr[1:3]    # 从下标 1 到 2，结果 [2,3]
arr[:2]     # [1,2]
arr[2:]     # [3]
arr[::-1]   # 反转
```

### tuple\( )（元组）——不可变List
不可变：不能修改内容
```Python
t = (1,2,3)
```

元组可以当作 **字典的 key**，list 不行。
例如：
```Python
mp[(0, 1)] = "point"
```
常用：
- BFS/DFS 存储坐标 `(x, y)`
- 函数返回多个值

### dict\{ , }（字典）——哈希表
创建
```Python
mp = {"a": 1, "b": 2}
```

访问：
```Python
mp["a"]   # 1
```

添加/更新：
```Python
mp["c"] = 3
```

检查 key 是否存在
```Python
"a" in mp
```

安全访问（推荐）
```Python
mp.get("x", 0)     # key 不存在时返回默认值 0
```

删除
````Python
mp.pop("a")
````

### set\{ }（集合）——无序、不重复、查找快
创建
```Python
s = {1,2,3}
```

添加
```Python
s.add(4)
```

删除
```Python
s.remove(2)
```

查找 O(1) 速度
```Python
if 3 in s:
    print("yes")
```

去重
```Python
s = set([1,2,2,3])   # {1,2,3}
```

### 容器之间的转换
```Python
list("abc")        # ['a', 'b', 'c']
"".join(['a', 'b', 'c']) # join是把列表里的元素连成一个字符串 →"abc"
set([1,2,2,3])     # {1,2,3} list→set
list({1,2,3})      # [1,2,3] set→list
tuple([1,2,3])     # (1,2,3) list→tuple
```


## 2.控制流（if / for / while）

### if 条件判断

```Python
if 条件:
    代码块
elif 另一个条件:
    代码块
else:
    代码块
```

### for 循环

遍历列表
```Python
arr = [10, 20, 30]

for x in arr:
    print(x)
```

```Python
nums = [10, 20, 30]

for i, num in enumerate(nums):
    print(i, num)
```

遍历字典
```Python
#遍历 key
for k in mp:
    print(k)
#遍历 key、value
for k, v in mp.items():
    print(k, v)
```

### while 循环（直到条件不满足才停）

```Python
i = 0
while i < 5:
    print(i)
    i += 1
```

```
0 1 2 3 4
```

### break（跳出循环）

```Python
for x in [1,2,3,4]:
    if x == 3:
        break
    print(x)
```

```
1
2
```

### continue（跳过本次循环）

```Python
for x in [1,2,3,4]:
    if x == 3:
        continue
    print(x)
```

```
1
2
4
```

## 4.字符串（string / str）

字符串就是 **不可变的字符序列**。
不可变：不能直接修改某个字符  
但可以用切片、拼接创建新的字符串。

```Python
a=1
str(a) #'1'

s = "hello"

s[0]   # 'h'
s[1]   # 'e'
s[-1]  # 'o'（倒数第一）

for c in s:
    print(c)

len(s) # 5

s = "abcdef"

s[1:4]   # 'bcd'
s[:3]    # 'abc'
s[3:]    # 'def'
s[::-1]  # 反转字符串：'fedcba'

a = "hello"
b = "world"
c = a + b   # 'helloworld'
```

### 字符串常用方法
```Python
# upper()/lower()大小写转换
"abc".upper()   # "ABC"
"HI".lower()    # "hi"

# strip()去除头尾空白
"  hi  ".strip()  # "hi"

# split()分割（文本处理最常用）
"a,b,c".split(",")   # ['a','b','c']
"hello world".split()# 按空格分割 ['hello', 'world']

# join()把列表拼成字符串
arr = ['a', 'b', 'c']
"".join(arr)     # 'abc'
",".join(arr)    # 'a,b,c'

"hello".replace("l", "x")  # 'hexxo'

# 查找某字符（in）
"a" in "abc"   # True
"z" in "abc"   # False
```

字符串是不可变的
```Python
s="hello"
# 用切片重新构造
s = 'x' + s[1:]  #xello
```

### ASCII、数字转换
```Python
# 字母转数字
ord('a')  # 97
ord('A')  # 65
ord('0')  # 48

# 数字转字符
chr(97)   # 'a'
chr(48)   # '0'
```

## 常用函数

### `len()` —— 求长度（所有容器通用）
```Python
len([1,2,3])      # 3
len("abc")        # 3
len({1,2,3})      # 3
len({"a":1})      # 1
```

### `range()` —— 生成整数序列（for 循环最常用）
```Python
range(5)        # 0,1,2,3,4
range(2, 6)     # 2,3,4,5
range(1,10,2)   # 1,3,5,7,9
range(i-1,-1,-1) # 从i-1开始，一直减到0（包含 0），每次减1
```

### `enumerate()` —— 遍历时带上索引
```Python
nums = [10, 20, 30]

for i, num in enumerate(nums):
    print(i, num)
```

```
0 10
1 20
2 30
```

### `sorted()` —— 排序（不改变原列表）
```Python
sorted([3,1,2])      # [1,2,3]
sorted("bca")         # ['a','b','c']

# 按 key 排序
sorted(words, key=len)

# 反向
sorted(arr, reverse=True)
```

### `sum()` —— 求和
```Python
sum([1,2,3])    # 6
```

### `max()` / `min()`
```Python
max([3,1,2])     # 3
min([3,1,2])     # 1

#带 key（如按长度）：
max(words, key=len)
```

### `zip()` —— 把多个列表“拉链式”组合
```Python
a = [1,2,3]
b = ['a','b','c']

list(zip(a,b))    # [(1,'a'), (2,'b'), (3,'c')]

# 常用于矩阵转置
matrix = [[1,2,3],[4,5,6]]
list(zip(*matrix))    # [(1,4), (2,5), (3,6)]
```

### `map()` —— 对每个元素执行函数
```Python
# 把所有数字变成字符串
list(map(str, [1,2,3]))   # ['1','2','3']
```

### `filter()` —— 按条件过滤
```Python
list(filter(lambda x: x>0, [-1,0,3]))
# [3]
```

### `any()` / `all()`
```Python
# 判断是否有一个为True：
any([False, True, False])   # True

#判断是否全部为True：
all([True, True, True])     # True
```

### `reversed()` —— 反转迭代器
```Python
list(reversed([1,2,3]))   # [3,2,1]
```

### `divmod()` —— 计算商和余数
```Python
divmod(17, 5)    # (3, 2)
```

### `abs()` —— 绝对值
```Python
abs(-10)    # 10
```
