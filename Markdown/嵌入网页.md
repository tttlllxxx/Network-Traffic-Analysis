要嵌入网页，在笔记中添加以下内容，并将占位符文本替换为您要嵌入的网页的 URL：

```html
<iframe src="INSERT YOUR URL HERE"></iframe>
```

## 嵌入 YouTube 视频 
***

要嵌入 YouTube 视频，请使用与[[基本格式化语法#外部图像|外部图像]]相同的 Markdown 语法：

```md
![](https://www.youtube.com/watch?v=u7x8RXwLKcA)
```
